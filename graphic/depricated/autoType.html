“Automotive Typography” Type Journal

Project Details

Client: Coursework

Medium: Digital Print

Tools: Adobe Illustrator, Adobe Photoshop, Adobe InDesign

Field: Graphic Design, Print Design, Typography, Editorial Design, photo editting

Objective: Compile a themed type journal displaying ten examples of typeforms for each letter of the alphabet. Must successfully create a cohesive theme, fully represent every letter, properly edit and package content for printing, and personally produce physical copy, including binding solution.
